Class Detail

  • Ajax (config, defData)
    Ajax方法
    参数：
    •   config {Object} 配置对象
    •   config.url {String} 发送请求的地址
    •   config.type {String} ["get" : "post"] 请求方式
    •   config.dataType {String} 请求方式
    •   config.dataType.xml {String} 返回 XML 文档，可用 jQuery 处理。
    •   config.dataType.text {String} 返回 String 字符串，可用 jQuery 处理。
    •   config.dataType.textJson {String} type自动转为text，返回数据预处理为 JSON 数据。
    •   config.dataType.html {String} 返回纯文本 HTML 信息；包含 script 元素。
    •   config.dataType.script {String} 返回纯文本 JavaScript 代码。不会自动缓存结果。
    •   config.dataType.json {String} 返回 JSON 数据 。
    •   config.dataType.jsonp {String} 使用 JSONP 形式调用函数时，如 "myurl?callback=?" jQuery 将自动替换 ? 为正确的函数名，以执行回调函数。
    •   config.queue {String} ajax队列的名称
    •   config.timeout {Number} 设置请求超时时间（毫秒）。此设置将覆盖全局设置。
    •   config.beforeSend {Function} 在发送请求之前调用，并且传入一个 XMLHttpRequest 作为参数
    •   config.success {Function} 当请求之后调用。传入返回后的数据，以及包含成功代码的字符串。
    •   config.error {Function} 在请求出错时调用。传入 XMLHttpRequest 对象，描述错误类型的字符串以及一个异常对象（如果有的话）
    •   config.complete {Function} 请求完成后回调函数 (请求成功或失败时均调用)
    •   defData {Object} 默认参数
    Since
    version 0.1
    示例：
    cgClass.Ajax({
    url : "api/ajax.txt",	
    data : {},	
    dataType : "textJson",	
    queue : "ajaxQueue",	
    success : function(){},	
    error : function(){}	
}); 
cgClass.AjaxQueueCallback.ajaxQueue = function(){}
